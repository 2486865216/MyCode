#纯函数和高阶函数
##纯函数
    一类特别的函数:只要是同样的输入(实参),必定得到同样的输出返回)
        必须遵守以下一些约束
            1)不得改写参数数据
            2)不会产生任何副作用,例如网络请求,输入和输出设备
            3)不能调用 Date. now0或者 Math. random0等不纯的方法
        redux的 reducer函数必须是一个纯函数
##高阶函数
    理解:一类特別的函数
        1)情况1:参数是函数
        2)情况2:返回是函数
    2.常见的高阶函数←
        定时器设置函数
