1)结构体的所有字段在内存中是连续的

2)结构体是用户单独定义的类型，和其它类型进行转换时需要有完全相同的字段（名字、个数和类型)
type A struct{
    Num int
}
type B struct{
    Num int
}
func main(){
    var aA
    var bB
    a =A(b)//?I
    fmt.Println(a,b)
}

4)struct的每个字段上，可以写上一个tag,该tag可以通过反射机制获取，常见的使用场景就是序列号和反序列化。
